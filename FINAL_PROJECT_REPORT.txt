PROJECT SUMMARY:

The Secure AI Coding Assistant is a multi-agent security pipeline
that orchestrates three specialized AI agents using LangGraph:

1. THE BUILDER (Code Generation)
   - Generates production-ready code
   - Follows security best practices
   - No tools available (pure code generation)

2. THE RED TEAM HACKER (Offensive Security Review)
   - Exhaustively attacks generated code
   - Identifies OWASP Top 10 & SANS Top 25 vulnerabilities
   - Provides attack vectors and proof-of-concepts

3. THE SECURITY AUDITOR (Final Verdict)
   - Synthesizes Red Team findings with static analysis
   - Uses tool access for automated scanning
   - Issues APPROVED or ESCALATED verdicts
   - Provides remediation briefs for failed code

PIPELINE FLOW:
User Request → Builder → Hacker → Auditor → Verdict
     ↓
(If ESCALATED) → Back to Builder with remediation brief
     ↓
(If APPROVED) → Generate audit reports
